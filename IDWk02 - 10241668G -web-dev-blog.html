My Understading About HTML

Week 1

  • h1 Is Used For The Main Heading

  • After h1, The Rest Are Sub Headings

  • p Is Used For A New Paragraph

  • img src Is Used to Insert A Picture

  • img alt Is Used When The Picture Fails To Load And Shows The Text Instead

  • <div> Tag Divides Documents Into Sections

  • A Site Conists Of Header, Body & Footer

  • <a> Tag Allows Link Between Documents & Resources

Week 2

  • CSS Helps Make Our Page More Presentable

  • <nav> Is Used For Main Elements In The Page

  • The Property & Value Of The Selector Are In Curly Braces

  • Hexadecimal, RGB & RGBA Can Be Used To Style Our Texts

  • Different Fonts Can Be Used By Using font-family

Week 3

  • Using <form> To Create A Form And <input> Tag To Input Date, Text & Radio etc

  • Validations Are Used To Make Sure An Input Is Not Empty

  • Style Cascade Is Read From Top To Bottom, Where The Style Will Take Precedence

  • We Can Write Our Styles in Three Ways. Inline, Style Tag Or External Stylesheet

  • Media Query Is Used To Modify CSS Of A Site Depending On The Device With The @media, Min-Width & Max-Width

  • Relative Positioning Tweaks An Element's Position Without Affecting Its Surroundings

  • Wireframs Can Display Cotent Hierarchy And Create Key Functionality
